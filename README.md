# koa-vue-blog

## 简介
服务端采用node的koa2搭建，使用mysql2驱动连接mysql数据库

使用sequelize进行操作数据库，bcryptjs进行加密

使用koa/router生成路由，koa-bodyparser解析post数据

ejs搭配koa-views做前端模板渲染，koa-static处理前端静态资源

前端使用vue+element+axios，当前版本还未使用webpack打包工具

### 执行以下命令
```
node app.js
```
